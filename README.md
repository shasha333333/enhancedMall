# enhancedMall

## 项目简介

本项目是一个基于Vue2 + Node.js + Mysql的商城项目

### 前端项目运行

```
npm run install         安装依赖
npm run serve           运行
npm run build           打包
```

### 后端项目运行
npm init -y
npm install express jwt-simple body-parser cors
node app.js             运行


### 数据库

server/mysql.js         数据库连接
